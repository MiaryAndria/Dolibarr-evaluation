Les principaux modules de Snipe-IT sont :

1. Assets (Actifs)

C'est le module principal.

Permet de gérer :

Ordinateurs
Laptops
Écrans
Smartphones
Serveurs
Imprimantes
Tout équipement informatique

Fonctionnalités :

Asset Tag
Numéro de série
Statut
Attribution à un utilisateur
Historique complet
Maintenance
2. Users (Utilisateurs)

Gestion des employés :

Création d'utilisateurs
Attribution d'actifs
Attribution de licences
Attribution d'accessoires
Gestion des permissions
Groupes
3. Licenses (Licences)

Gestion des licences logicielles :

Exemples :

Microsoft Office
Windows
Adobe Photoshop
JetBrains

Fonctionnalités :

Clé de licence
Date d'expiration
Nombre de sièges
Attribution aux utilisateurs
4. Accessories (Accessoires)

Matériel distribué aux utilisateurs :

Exemples :

Souris
Clavier
Casque
Webcam
Dock USB

Contrairement aux Assets, ils ne sont généralement pas suivis individuellement.

5. Consumables (Consommables)

Articles consommables :

Exemples :

Cartouches d'encre
Piles
Papier
Câbles jetables

Gestion :

Stock
Distribution
Historique
6. Components (Composants)

Pièces installées dans des équipements :

Exemples :

RAM
SSD
HDD
Carte graphique
Processeur

Permet de savoir quels composants sont installés dans quels actifs.

7. Locations (Emplacements)

Gestion des sites :

Exemples :

Antananarivo
Bureau Paris
Datacenter

Permet d'affecter :

Assets
Utilisateurs
Consommables

à un lieu précis.

8. Manufacturers (Fabricants)

Exemples :

Dell Technologies
HP Inc.
Lenovo
Apple
9. Suppliers (Fournisseurs)

Entreprises auprès desquelles le matériel est acheté.

Exemples :

Revendeur local
Grossiste informatique
10. Categories (Catégories)

Classification des éléments :

Exemples :

Laptop
Desktop
Téléphone
Imprimante
11. Asset Models (Modèles)

Modèles d'équipements :

Exemples :

Dell Latitude 5520
HP EliteBook 840
Lenovo ThinkPad T14
12. Status Labels (Statuts)

Définissent l'état d'un actif :

Ready to Deploy
Deployed
Pending
Archived
Broken
13. Reports (Rapports)

Rapports et exportations :

Inventaire complet
Licences expirées
Actifs attribués
Historique

Export :

CSV
Excel
14. Settings (Paramètres)

Réservé principalement aux Super Admins.

Contient :

Configuration générale
Email
LDAP
SAML
API
Branding
Notifications
Sauvegardes
Sécurité
